## About This Repository

This repository contains the **HTML and CSS** for my personal website. It is hosted on **GitHub Pages** and automatically updates whenever changes are pushed to the `main` branch.

### Structure

- `index.html` — Main HTML file, contains the website structure  
- `style.css` — Main CSS file, contains styling and layout  
- `CNAME` — Custom domain configuration for `ryanmccuaig.ca`  
- `.github/workflows/pages-deploy.yml` — GitHub Actions workflow for automatic deployment
